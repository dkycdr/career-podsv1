## Interaction States

### 1. Idle State
- Input field fokus
- Send button aktif
- Clear chat button aktif

### 2. Sending State
- Input field disabled
- Send button disabled
- Loader running
- Last message: "Mengetik..."

### 3. Error State
- Bot message: "Maaf, ada error. Silakan coba lagi."
- Input field kembali fokus
- Send button aktif

### 4. Empty Chat State
- Initial greeting dari bot
- Input field kosong dan fokus
- Clear chat button visible
